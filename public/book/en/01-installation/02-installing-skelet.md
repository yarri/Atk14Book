Installing a skeleton
=====================

Once you have met the requirements from the previous chapter, you are ready to install one of the available skeletons and build your great new project on top of it.

A skeleton is a small, fully functional ATK14 application that you can conveniently extend and adapt to your needs. The skeleton source code is open, versioned on GitHub, and the ATK14 Framework itself is included as a submodule.

There are currently five skeletons available.

### 1. ATK14 Embryo

This is a blank slate. The absolute minimum. One page and one fancy image. ATK14 Embryo is a skeleton for projects that break every convention ever seen. You'll have to build everything yourself.
Good luck with that.

* live demo: <http://embryo.atk14.net/>
* source code: <https://github.com/atk14/Atk14Embryo.git>
* installation guide: <https://github.com/atk14/Atk14Embryo/blob/master/README.md#installation>

Minimalist, isn't it?

### 2. ATK14 Mini Skelet

This skeleton was created by slimming down ATK14 Skelet, which had grown quite large during development. Administrable pages, articles, link lists, and other components were removed. However, the important building blocks for a future application were kept, such as support for multilingual text in application models, slug handling for beautiful URLs, etc.

* live demo: <http://miniskelet.atk14.net/>
* source code: <https://github.com/atk14/Atk14MiniSkelet.git>
* installation guide: <https://github.com/atk14/Atk14MiniSkelet/blob/master/README.md#installation>

### 3. ATK14 Skelet

You will love this one. ATK14 Skelet contains a set of well-tuned things that simply work.

* live demo: <http://skelet.atk14.net/>
* source code: <https://github.com/atk14/Atk14Skelet.git>
* installation guide: <https://github.com/atk14/Atk14Skelet/blob/master/README.md#installation>

### 4. ATK14 Catalog

ATK14 Catalog is based on ATK14 Skelet, extended mainly with a _product catalogue_.

* live demo: <http://catalog.atk14.net/>
* source code: <https://github.com/atk14/Atk14Catalog.git>
* installation guide: <https://github.com/atk14/Atk14Catalog/blob/master/README.md#installation>

### 5. ATK14 Eshop

ATK14 Eshop is based on ATK14 Catalog, extended mainly with a full-featured e-shop.

* live demo: <http://eshop.atk14.net/>
* source code: <https://github.com/atk14/Atk14Eshop.git>
* installation guide: <https://github.com/atk14/Atk14Eshop/blob/master/README.md#installation>

### Summary

It can be hard to pick the right skeleton from so many options. If you're not sure, try them all :)
